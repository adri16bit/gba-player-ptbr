ℹ RESUMO

GBA mGBA e um emulador de Game Boy Advance para Google Chrome com core mGBA. Esta extensao nao hospeda jogos e nao distribui ROMs.

Suas ROMs, saves e configuracoes ficam apenas no seu navegador. Nao coletamos nome real, e-mail, localizacao nem dados pessoais identificaveis.

Ultima atualizacao: julho de 2026

🎮 FUNCIONALIDADES E DADOS

Cada recurso da extensao e o que acontece com seus dados:

Importar e jogar ROMs .gba SÓ LOCAL
Biblioteca de ROMs SÓ LOCAL
Save de cartucho (.sav) SÓ LOCAL
Auto Save SÓ LOCAL
Salvar / carregar estado SÓ LOCAL
Controles de teclado SEM DADOS
Som e volume SÓ LOCAL
Tela cheia SEM DADOS
Idioma PT / EN SÓ LOCAL
Core mGBA (WASM) SÓ LOCAL
Ranking mundial de horas SUPABASE

A emulacao roda 100% no seu dispositivo. Nenhuma ROM, save ou gameplay e enviado para servidores.

📊 DADOS ENVIADOS (RANKING)

Apenas para o ranking mundial opcional, enviamos estatisticas anonimas:

  • ID anonimo — gerado na instalacao (ex.: Player A3F2)
  • Tempo total de jogo — em milissegundos
  • Contagem de jogadores — total de usuarios unicos

Destino: Supabase (servidor nos EUA). Sem ROMs, sem saves, sem historico de navegacao.

💾 DADOS LOCAIS (NAO ENVIADOS)
  • ROMs importadas (.gba)
  • Saves de cartucho (.sav)
  • Save states do emulador
  • Preferencias (idioma, volume, som)
  • Lista da biblioteca de jogos

Tudo fica em chrome.storage.local no seu Chrome.

🔐 PERMISSOES DO CHROME

storage

Salvar ROMs, saves, configuracoes e ID anonimo localmente.

unlimitedStorage

ROMs e saves podem ocupar muito espaco. Sem isso, o limite padrao do Chrome impediria guardar varios jogos.

https://*.supabase.co/*

Usado somente para sincronizar o ranking mundial anonimo descrito acima.

🤝 TERCEIROS

Nao vendemos dados. Nao usamos anunciantes. Apenas o Supabase processa as estatisticas anonimas do ranking.

⚖ SEUS DIREITOS

ℹ SUMMARY

GBA mGBA is a Game Boy Advance emulator for Google Chrome powered by the mGBA core. This extension does not host games and does not distribute ROMs.

Your ROMs, saves, and settings stay only in your browser. We do not collect real names, emails, location, or personally identifiable information.

Last updated: July 2026

📊 DATA SENT (RANKING)

Only for the optional global ranking, we send anonymous statistics:

  • Anonymous ID — generated on install (e.g. Player A3F2)
  • Total play time — in milliseconds
  • Player count — total unique users

Destination: Supabase (US servers). No ROMs, no saves, no browsing history.

💾 LOCAL DATA (NOT UPLOADED)
  • Imported ROMs (.gba)
  • Cartridge saves (.sav)
  • Emulator save states
  • Preferences (language, volume, sound)
  • Game library list

Everything stays in chrome.storage.local in your Chrome.

🔐 CHROME PERMISSIONS

storage

Store ROMs, saves, settings, and local anonymous ID.

unlimitedStorage

ROMs and saves can use a lot of space. Without this, Chrome's default quota would prevent storing multiple games.

https://*.supabase.co/*

Used only to sync the anonymous global ranking described above.

🤝 THIRD PARTIES

We do not sell data. We do not use advertisers. Only Supabase processes the anonymous ranking statistics.
